# Rock Paper Scissors - Python Game

## Project Structure

```
├── main.py          # Entry point — runs the game
├── game_logic.py    # Core rules: who beats whom, result messages
├── computer.py      # Computer's random move selection
├── player.py        # Player input and validation
├── scoreboard.py    # Score tracking and display
└── README.md        # This file
```
## Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Tie
